Loose Search Strict search

  • This tool performs a deep-web search based on the user's input and then redirects it to a google website listing some of the domains that have stored information about the search query in the format selected by the user and all the data shown is indexed and gives access to hidden databases for research purposes.

  • To use this tool just enter your research topic in the input field and the select the format from drop-down option and click on Go! And your are done all your content would be listed on the next website on which you were redirected.

  • This tool is made by co-operation of two aspiring developers.

  • Piyush Mishra (From India) did the frontend and ODFinder (From Russia) did work on the algorithms for deep-search queries.
